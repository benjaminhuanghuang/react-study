##  React Unit Test Toolkit

## Jest 
    Jest is a JavaScript test framework based on Jasmine. 
    Jest has following key features 
    - DOM API
    - Assertion
    - Mock
    - Code coverage
    - Snapshot testing
    - Instant Feedback
    
## Enzyme
    jQuery style DOM API

## Testing Framework
- Jest - Experimental support for ECMAScript Modules
- Mocha
- Jasmine
- Vitest - Supports ESM, TypeScript, and JSX
```sh
npm i -D vitest

Install Vitest Snippets VS code extension

# package.json
"test": "vitest",
"test:ui": "vitest --ui"
```
